# Интернет-магазин канцелярских товаров

**Тема:** Разработка сайта отдела по продаже канцелярских товаров  
**Цель:** Создать специализированный интернет-магазин для онлайн-продаж канцелярских товаров

## Ссылки
- Сайт: [канцелярия.рф](http://канцария.рф)
- Прототипы в Figma: [Открыть проект](https://www.figma.com/design/5RvbPwxwPNO4BAFMoJzEmZ/d?t=zZ1Jhafkpf6B8Spb-0)

## Технологии
- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP 7.4+
- База данных: MySQL 5.7+
- Хостинг: Masterhost

## Целевая аудитория
Школьники и студенты 15–25 лет
